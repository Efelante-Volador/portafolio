  • Pagina web "Casita Latina"

    Este proyecto consistió en una página web informativa y responsiva para una micro-empresa de idiomas, Christian el cliente, es un argentino que se fue a vivir a Polonia, hoy en día trabaja enseñando español en este país, necesitaba un sitio web donde pudiera promocionar sus servicios y dar a conocer su información de contacto. Su sitio web Casita Latina se encuentra completamente en Polaco. este sitio se realizo con HTML, CSS (Boostrap), y JavaScript

  • Bot de Discord

    Discord es una aplicación que permite comunicarse a través de mensajes de texto, voz y video diseñada principalmente para comunidades de videojuegos. Se programó un Bot para leer mensajes de texto y reaccionar a ellos mediante ciertos comandos. Estos comandos utilizaban un apóstrofe o palabra clave para interpretarlos como comandos. La aplicación se realizó en Replit, una plataforma de programación que permite ejecutar código en diferentes lenguajes. Se mantuvo en línea mediante un pinger o herramienta de ping que verificaba si el sitio estaba en línea cada 5 minutos. Se utilizó Node js como entorno de ejecución, algunas de las acciones que realizaba este bot era, la asignacion de roles a los usuarios, dar hora Utc y dar la cantidad de usuarios conectados o usuarios dentro de la comunidad.
